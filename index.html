⚙ Configuración EmailJS (una sola vez)

EmailJS permite enviar correos desde el navegador sin servidor. Es gratis hasta 200 correos/mes.
Pasos: emailjs.com → Crear cuenta → Add Service (Gmail/Outlook) → Create Template → copiar IDs abajo.

En el template de EmailJS usa estas variables: {{to_email}} {{subject}} {{message}}
